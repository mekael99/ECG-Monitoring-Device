# ECG Monitoring Device

A custom-built ECG device that captures 
and displays real-time cardiac signals.

## Hardware
- AD8232 ECG Sensor Module
- LilyGO TTGO T-Display (ESP32)

## Features
- Real-time ECG signal capture
- Live signal display on TFT screen
- Full signal acquisition and 
  processing pipeline

## Technologies
- Embedded Systems
- Electronics
- Signal Processing
- C++ / Arduino
